# Glitch.com Deploy (Free, No GitHub needed)

## Steps:

1. https://glitch.com போங்க
2. "Sign in" → Google/Email login
3. "New Project" → "glitch-hello-node" click
4. Left side file list-ல் `server.js` click
5. எல்லாத்தையும் delete பண்ணி → tn-cloud/server.js content paste
6. `package.json` click → replace with tn-cloud/package.json content
7. Top-ல் உங்க project name காட்டும் (e.g. "rosy-pickle")
8. URL: https://rosy-pickle.glitch.me/view/082%2F001%2F900540

Done! ✅
